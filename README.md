# AnimeNews - MJV

Projeto desenvolvido durante a Devschool. É uma página que apresenta notícias sobre animes.


## Tecnologias usadas:

- Java 8
- Spring MVC 5.2.9
- Spring Boot 2.3.4
- Spring JBDC
- H2 database
- JSP
- Bootstrap 4.5.3
